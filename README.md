1. 项目概述
公司名称：四川井明电子有限公司（Sichuan Kinming Electronics Co., Ltd.）

行业：PCB（印制电路板）制造，提供 2-24 层 PCB，应用于航空、通讯、汽车、AI、医疗等领域。

官网类型：企业展示型网站，单页应用风格（前端路由 + 锚点滚动），无需后端数据库。

核心要求：

顶部导航栏带二级下拉菜单。

每个一级页面顶部有自动轮播图（可配置图片、标题、描述，支持中英文）。

页面内容包括公司介绍、行业市场、产品技术、新闻中心、社会责任、联系我们等模块。

底部有“社会责任声明”（绿色制造）以及多列链接导航（与顶部导航结构一致）。

全站支持中英文切换。

公司地址已固定：四川省遂宁市经济技术开发区渠河南路188号电子电路标准厂房一期3号楼。

4. 交互行为
点击一级菜单：切换页面内容（隐藏其他页面，显示当前页面），URL hash 更新（如 #about），页面顶部滚动到轮播图。

点击二级菜单（下拉或底部链接）：

如果目标页面不是当前页面 → 先切换到目标页面 → 等待页面渲染完成 → 滚动到该二级菜单对应的子区块（通过锚点 ID）。

如果已在目标页面 → 直接平滑滚动到该子区块。

语言切换：所有文本（导航、轮播图文案、页面内所有静态文本、底部声明、链接标题等）中英文互换。无需刷新页面，使用 JavaScript 动态替换文本内容。

轮播图：自动循环 + 点击指示点切换，切换语言时轮播图文字同步更新（若图片内容依赖语言，则重新渲染轮播图组件）。

5. 多语言数据要求
中英文两套完整文本库，结构一致。

导航标题、各页面所有描述、表格、列表、底部声明、公司地址等均需提供英文版本。

轮播图每页的中英文标题/描述独立存储。
